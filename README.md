### Installing and running the project:

	1- Clone this repository
	
		git clone https://github.com/Kuninoto/Libft
	2- `cd` `make`
	
		cd
    	make
	3- Compile your main.c with `libft.a`
  
    gcc -Wall -Wextra -Werror main.c libft.a
